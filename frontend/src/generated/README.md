# Generated product content

The contents of this directory are regenerated by
`scripts/generate_products.py`. Do not edit by hand.

- `products.json` — flat summary list consumed by the router (for SSG
  `includedRoutes`) and by `ProductsIndexView` for filtering/sorting.
- `products/<slug>.json` — per-course full snapshot (stats, AI copy,
  asset paths) imported dynamically by `ProductView`.
